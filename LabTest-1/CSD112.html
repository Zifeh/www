Introduction to Web Development

In this course CSD112 you will learn the fundamentals of technology that is used to create HTML and CSS for websites. You will be taught all the elements used to create a functioning webpage, some element tags in html include; div,p1,head,main, etc. Some topics you will be covering in this course are ;
  • What is CSS and how is it used
  • How to create custom layouts for a HTML webpage
  • How to use Github, modern web browsers and Visual studio code to create working HTML websites
